twentefp-eventloop-graphics-0.1.0.5: Used as Lab Assignments Environment at Univeriteit Twente

An eventloop based graphical IO system. It uses websockets as a way to communicate with IO devices; a browser in this case. This system is used in the Graphical submodule to be able to express graphical output using the eventloop system for a browser.
