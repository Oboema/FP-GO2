retrieve :: [Char] -> [JSONMember] -> Maybe JSONMessage Source

Retrieves the field content (JSONMessage) associated with the field name ('[Char]') from a list of JSONMembers.

retrieveError :: [Char] -> [JSONMember] -> JSONMessage Source

Same as retrieve but instead of returning a Maybe, it raises an error when the field name could not be found. Usable in situations where you know for sure the field name is in the list of JSONMembers.
